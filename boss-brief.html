Problem & Solution

Manual Process Today

  • Log into S-Cubed every week manually
  • Remember what was worked on each day
  • Manually enter client, project & hours
  • Risk of late or incomplete submissions

Automated with This Tool

  • Open app, click Submit — done in 30 sec
  • Reads calendar, git history & work sessions
  • Auto-fills all fields with accurate data
  • Preview screen before anything is saved

Download & Install Instructions
Windows
  1. Download .exe
  2. Double-click ScubedTimesheet-windows.exe
  3. If Defender shows a warning → More info → Run anyway
  4. Click First-time Setup and log in with Okta
macOS
  1. Download binary
  2. Open Terminal in the download folder
  3. xattr -d com.apple.quarantine ScubedTimesheet-macos
  4. chmod +x ScubedTimesheet-macos && ./ScubedTimesheet-macos
  5. Click First-time Setup and log in with Okta

Or right-click → Open → Open to skip step 3.

Linux
  1. Download binary
  2. Open Terminal in the download folder
  3. chmod +x ScubedTimesheet-linux
  4. ./ScubedTimesheet-linux
  5. Click First-time Setup and log in with Okta
First-time setup is once only. After logging in with Okta, the app runs fully headless — no browser window, no manual steps. Every week after: open the app → click Preview to review → click Submit Timesheet.
